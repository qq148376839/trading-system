## 📊 项目概览

### 项目名称
长桥股票交易系统（Trading System）

### 项目类型
全栈量化交易平台

### 技术栈
- **后端**: Node.js + Express + TypeScript + PostgreSQL
- **前端**: Next.js 14 + React + TypeScript + Tailwind CSS
- **交易API**: Longbridge SDK（长桥证券）
- **市场数据API**: Moomoo API（富途牛牛）
