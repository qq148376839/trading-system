# 盘中实时监控大屏

**日期**: 2026-04-04
**路由**: `/quant/monitor`

## 功能概述

科技感暗黑主题的盘中监控聚合页面，一屏展示市场评分、策略状态、信号流、今日P&L。

## 页面布局

Desktop 四个区块，移动端纵向堆叠:

1. **市场评分** — SVG 仪表盘 + AreaChart 趋势图 + 评分子项 badges + 总分中文解读 + 市场环境类型 + 适用策略推荐
2. **策略全局状态** — 所有运行策略的实例 badges（状态/P&L）
3. **信号流** — 最近 20 条信号（时间/标的/方向/得分/状态）
4. **今日P&L** — 已实现/未实现/总计 + 分策略明细

## 后端 API

### `GET /quant/monitor/market-score`

返回当前市场评分完整数据:
- `finalScore` (-100~+100) + `scoreLabel` (中文解读)
- `marketScore` / `intradayScore` / `timeWindowScore` 三个子分
- `marketComponents` — 9 个子项明细（SPX日/分, USD日/分, BTC日/分, VIX, 温度, Gap）
- `direction` (CALL/PUT/HOLD) + `confidence`
- `regime` — 市场环境类型 (MOMENTUM/MEAN_REVERSION/UNCERTAIN) + 置信度
- `suitableStrategies` — 当前适用策略类型推荐（中文）

**数据来源**: `marketDataCacheService` 30s 缓存 + SPX 5min K线 + `marketRegimeDetector`

### `GET /quant/monitor/strategies-overview`

聚合所有 RUNNING 策略:
- 策略名称、类型
- 每个实例: symbol/state/entryPrice/stopLoss/takeProfit/unrealizedPnl/cooldownUntil
- 实例状态汇总: idle/holding/opening/closing/cooldown 计数
- 当日已实现 P&L + 交易笔数

## Polling 策略

| 数据 | 间隔 | 说明 |
|------|------|------|
| 市场评分 | 30s | 复用缓存，无额外 API 调用 |
| 策略状态 | 10s | 仅查数据库，不调 Longport SDK |
| 信号+统计 | 30s | 复用现有端点 |

评分历史: 前端 state 累积（每 30s 追加，最多 240 个点 = 2 小时）

## 主题设计

- 页面级作用域（`.monitor-page` 容器）
- 背景: `#0a0f1c` / 卡片: `rgba(16, 24, 48, 0.8)`
- 强调色: `#00d4ff`(氰) / `#00ff88`(绿) / `#ff4757`(红)
- 数字: monospace + tabular-nums
- 动效: 呼吸灯 LIVE 指示器、发光边框
