# 批量重命名脚本说明

由于文件数量较多，建议使用以下PowerShell脚本批量重命名：

## PowerShell脚本

```powershell

# guides目录重命名

Rename-Item -Path (Join-Path $guidesDir "251212-TRADING_GUIDE.md") -NewName "251212-交易功能使用说明.md"
Rename-Item -Path (Join-Path $guidesDir "251214-DOCKER_DEPLOYMENT_SUMMARY.md") -NewName "251214-Docker部署完成总结.md"
Rename-Item -Path (Join-Path $guidesDir "251214-DOCKER_SETUP.md") -NewName "251214-Docker环境设置指南.md"

$technicalDir = Join-Path $baseDir "docs\technical"
```

## 根目录文件移动脚本

```powershell
# 设置工作目录
$baseDir = "D:\Python脚本\trading-system"
$archiveDir = Join-Path $baseDir "docs\archive\2025-12-16-analysis"
$guidesDir = Join-Path $baseDir "docs\guides"

# 创建归档目录（如果不存在）
if (-not (Test-Path $archiveDir)) {
    New-Item -ItemType Directory -Path $archiveDir -Force
}

# 移动分析报告到归档目录
$filesToMove = @{
    "ANALYSIS_SUMMARY_UPDATE.md" = "251216-交易策略分析总结更新.md"
    "analyze_backtest_logic_final.md" = "251216-回测交易逻辑分析报告.md"
    "CHECK_ORDERS_DATA.md" = "251216-订单数据检查.md"
    "BACKFILL_MATCHING_ANALYSIS.md" = "251216-回填匹配失败分析.md"
    "CLEANUP_EXECUTION_RESULT.md" = "251216-数据清理执行结果.md"
    "CLEANUP_SUCCESS_SUMMARY.md" = "251216-数据清理成功总结.md"
    "BACKFILL_ANALYSIS.md" = "251216-回填脚本执行结果分析.md"
    "DATA_CLEANUP_PLAN.md" = "251216-历史数据清理方案.md"
    "HISTORICAL_ORDERS_SIGNAL_CREATION.md" = "251216-历史订单信号创建方案.md"
    "OPTIMIZATION_IMPLEMENTATION.md" = "251216-交易策略优化实施总结.md"
    "SIGNAL_CREATION_SUCCESS.md" = "251216-历史订单信号创建成功总结.md"
    "TRADING_STRATEGY_ANALYSIS_REPORT.md" = "251216-交易策略问题分析报告.md"
    "WARNING_LOGS_ANALYSIS_REPORT.md" = "251216-警告日志分析报告.md"
}

foreach ($file in $filesToMove.GetEnumerator()) {
    $sourcePath = Join-Path $baseDir $file.Key
    $destPath = Join-Path $archiveDir $file.Value
    if (Test-Path $sourcePath) {
        Move-Item -Path $sourcePath -Destination $destPath -Force
        Write-Host "已移动: $($file.Key) -> $($file.Value)"
    }
}

# 移动指南文档到guides目录
$guidesToMove = @{
    "ENV_SETUP_GUIDE.md" = "251216-环境变量配置指南.md"
    "FRONTEND_API_URL_SETUP.md" = "251216-前端API-URL配置指南.md"
}

foreach ($file in $guidesToMove.GetEnumerator()) {
    $sourcePath = Join-Path $baseDir $file.Key
    $destPath = Join-Path $guidesDir $file.Value
    if (Test-Path $sourcePath) {
        Move-Item -Path $sourcePath -Destination $destPath -Force
        Write-Host "已移动: $($file.Key) -> $($file.Value)"
    }
}

# 移动清理总结文档到archive目录
$cleanupFiles = @(
    "DOCUMENTATION_CLEANUP_SUMMARY.md",
    "DOCUMENTATION_CLEANUP_SUMMARY_20251216.md",
    "DOCUMENTATION_CLEANUP_PLAN_20251216.md",
    "DOCUMENTATION_CLEANUP_MANUAL.md"
)

$archiveRoot = Join-Path $baseDir "docs\archive"
foreach ($file in $cleanupFiles) {
    $sourcePath = Join-Path $baseDir $file
    if (Test-Path $sourcePath) {
        Move-Item -Path $sourcePath -Destination $archiveRoot -Force
        Write-Host "已移动: $file -> docs/archive/"
    }
}

Write-Host "文件移动完成！"
```

## Docker文档重命名脚本

```powershell
# 设置工作目录
$baseDir = "D:\Python脚本\trading-system"

# Docker文档重命名（保留在根目录）
$dockerFiles = @{
    "DOCKER_BUILD_FIX.md" = "Docker构建问题修复说明.md"
    "DOCKER_MIRROR_FIX.md" = "Docker镜像拉取超时问题解决方案.md"
    "DOCKER_OPTIMIZATION.md" = "Docker配置优化说明.md"
    "DOCKER_TROUBLESHOOTING.md" = "Docker部署故障排查指南.md"
}

foreach ($file in $dockerFiles.GetEnumerator()) {
    $sourcePath = Join-Path $baseDir $file.Key
    $destPath = Join-Path $baseDir $file.Value
    if (Test-Path $sourcePath) {
        Rename-Item -Path $sourcePath -NewName $file.Value -Force
        Write-Host "已重命名: $($file.Key) -> $($file.Value)"
    }
}

Write-Host "Docker文档重命名完成！"
```
