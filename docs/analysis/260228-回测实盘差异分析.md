# 回测 vs 实盘差异分析（2/27 盘后复盘）

**日期**: 2026-02-28
**数据**: 回测 +$278 (33% WR, 2/6) vs 实盘 -$16 (43% WR, 3/7)，差距 ~$294

## 差异根因

### 1. 策略启动时间不同
- 回测：9:54 ET 开始交易
- 实盘：11:21 ET 才启动策略
- 影响：错过早盘最佳信号窗口

### 2. 合约选择逻辑不同
- 回测：`roundToStrike()` 算术取整（TSLA $2.5 间隔 → 固定 P402.5）
- 实盘：Longbridge API 实时期权链 → 流动性排序 → 8 个 near-ATM strike 中选最优（→ P400）
- 影响：不同合约的 gamma/delta 特性差异导致盈亏幅度不同

### 3. SPY 交易来源确认
- 回测未生成 SPY 信号，但实盘执行了 SPY260224P686000
- 结论：非数据错误，是 R5v2 竞价机制从候选池中选出 SPY（当其他标的评分不足时）

### 4. 持仓时间极短
- 实盘多笔持仓仅 16 秒即平仓
- 原因：peakPnLPercent 跨交易继承 → 虚假尾部止损（已在 b188cdb 修复）

### 5. 限价单滑点
- ELO 限价单执行显示严重滑点，实际成交价偏离预期

## 结论

差异中约 60% 可归因于 peakPnLPercent 跨交易继承 bug（已修复），剩余 40% 来自合约选择路径差异和启动时间差异（结构性差异，需长期优化）。
