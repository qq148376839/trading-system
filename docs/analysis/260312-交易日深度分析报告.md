### 3.3 发现的潜在问题

**问题 1：入场金额不均匀，资金利用率差异大**

| 交易 | 入场成本(premium x 100 x qty) | 占比 |

| NVDA 180P x 2 | $206 | 中等 |
| ORCL 160P x 1 | $185 | 较低 |
| TSLA 390P #1 x 1 | $242 | 中等 |
| TSLA 390P #2 x 1 | $271 | 较高 |

金额范围 $144 - $271，虽然大致在 $200 附近，但 QQQ 597P 只分配了 $144，而 TSLA 第2笔高达 $271。在 MAX_PREMIUM 模式下，这说明 `budget` 是固定的，但不同期权单价不同导致合约数不同 -- **单价低的标的用不完预算，单价高的标的刚好用完**。

**问题 2：TSLA 快速重入场 -- 第2笔买在更高价**

- TSLA #1: 14:42:36 买入 $2.42 -> 14:56:03 卖出 $2.64 盈利 +$22
- TSLA #2: 14:56:20 买入 $2.71 -> 15:39:59 卖出 $1.62 亏损 -$109

第2笔在第1笔平仓后仅 **17秒** 就重新入场，且入场价比第1笔高 $0.29（+12%）。这意味着：
- 策略在 TSLA 刚盈利退出后立即产生了新信号
- 新信号的入场价更高，说明 Put 在瞬间变贵了（标的继续下跌）
- 但随后标的反弹，Put 大幅贬值 -> 产生当日最大亏损

**根因**：非0DTE 冷静期（`zdteCooldownMinutes`）可能设置过短或未生效，允许了同标的快速重入。

**问题 3：QQQ 连续亏损重入**

- 但底层风险暴露是相同的（同一 underlying QQQ）
