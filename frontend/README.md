# 前端应用

Next.js + TypeScript + Tailwind CSS 实现的前端应用。

## 功能

- 实时行情展示
- 关注股票管理
- K线图表分析
- 交易记录查询

## 开发

```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:3000

## 环境变量

创建 `.env.local` 文件：

```bash
NEXT_PUBLIC_API_URL=http://localhost:3001
```

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Axios
- Recharts (图表库)
